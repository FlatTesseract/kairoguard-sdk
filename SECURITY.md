# Security Policy

## Supported Versions

Security fixes are provided for:

- Latest published version of `@kairoguard/sdk`
- Previous minor release line when feasible

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue.

- Email: `security@kairoguard.com`
- Subject: `Kairo SDK security report`
- Include: affected version, impact, proof of concept, and reproduction steps

We will acknowledge receipt within 72 hours and provide status updates as the investigation progresses.

## Disclosure Process

- We validate and triage the report
- We develop and test a fix
- We coordinate a release
- We publish a security advisory and mitigation guidance when appropriate

Please give us reasonable time to remediate before public disclosure.
