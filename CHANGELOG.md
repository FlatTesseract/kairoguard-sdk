# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.16] - 2026-03-22

### Added

- Sui Move contracts (`kairo_policy_engine`, `kairo_governance`) added to public repo.
- Stripped hackathon-edition backend with core signing, policy, and vault routes.
- Architecture and security documentation for judge review.

## [0.0.13] - [0.0.15]

Incremental fixes and improvements released from the private monorepo (rate-limit handling, RPC proxy resilience, SDK activation polling, signature recovery, ESM import fixes). These versions were published to npm before the public repo was established.

## [0.0.12] - 2026-03-19

### Added

- Public standalone repository setup for `@kairoguard/sdk`.
- Open source governance files and GitHub automation.

### Notes

- Initial open source baseline extracted from private monorepo.
